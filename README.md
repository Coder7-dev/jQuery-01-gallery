Название: Галерея изображений "Мстители"
Ссылка: https://github.com/Coder7-dev/jQuery-01-gallery
Стек: HTML, CSS, JS (jQuery)

Задачи: 
Реализовал классическую интерактивную галерею с управлением состояниями с помощь библиотеки jQuery. 

Фичи:
1. Запрограммировал скрывающуюся панель контента с плавной вертикальной анимацией и динамическим переключением текста на кнопке управления в зависимости от состояния панели.
2. Реализовал логику замены главного изображения при клике на миниатюру и добавил проверку, чтобы защитить интерфейс от лишних перезапусков анимации при повторном клике.
3. Реализовал систему визуального фокуса для миниатюр галереи: настроил динамическое управление прозрачностью и CSS-границами элементов для выделения выбранного медиа-объекта, чтобы помочь пользователю легко ориентироваться в галерее.
